Full 90s Chaos is a retro-styled, multi-page website that analyzes the early-2000s virtual world Neopets through the lenses of community, economy, daily gameplay loops, and long-term digital legacy. The project blends archival research, scholarly analysis, and intentionally chaotic 1990s web aesthetics to recreate the feel of early fan-made Neopets and Geocities webpages.

- Demonstrate thoughtful engagement with both primary web artifacts and academic arguments.

## Key Features
- **Animated cursor sparkles:** Every mouse movement generates Neopian-style spark effects.
- **Magical hover links:** Gradient underline, sparkle flash, and gold hover color (with optional rainbow glow).
- **Archival integration:** Primary sources from 1999-2001 (via the Wayback Machine) embedded or referenced throughout.
- **Modular pages:** Each page focuses on a separate argument but keeps a unified visual style.

## Reflection 
This project explores how early virtual worlds shaped children's digital literacy, economic behavior, and socialization long before modern social media. Recreating the aesthetic of the era adds an experiential layer, showing how design, community, and commercial techniques worked together in Neopets. Combining archival material, academic sources, and period-accurate web design highlights the early internet as a cultural space, not just a technological one.
